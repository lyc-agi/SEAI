# SEAI
自我进化的人工智能 (Self-evolving Artificial Intelligence, SEAI)  
基于刘逸川的智能理论。

这个项目基于这样一种思路，即通用人工智能可以通过相对简单的初始状态和规则进化而来，虽然也可以直接设计通用人工智能，但是这相对复杂和困难，也不符合智能的初衷：提高效率。
